# Given a string S For each word in S reverse its letters then print it.

## Note: words are separated by space.

## Input
Only one line contains a strings S (1≤|S|≤106) where |S| is the length of the string and it consists of lowercase and uppercase English letters, spaces.

## Output
Print the answer required above.

Examples
## Input
I love you
## Output
I evol uoy
## Input
You love me
## Output
uoY evol em
## Input
We are a happy family
## Output
eW era a yppah ylimaf
